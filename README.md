# Voidarchy

Sleek Hyprland dotfiles for Arch Linux.
Minimal aesthetic, maximum performance.

## About

Voidarchy is a minimal Hyprland rice made mainly for Arch Linux users who want
a clean workflow and good performance without too much visual noise.

> This project is currently under development / ricing phase.
> What you see now is just the initial stage, things will change  over time.

## Focus points:
- minimal look
- fast startup
- simple and usable keybinds
- clean configs
- less bloat

> You may need to adjust few things depending on your hardware.

## Installation

Clone the repository:
```bash
git clone https://github.com/jigarvarma2k20/voidarchy.git
cd voidarchy
```
Run the installer:
```bash
chmod +x install.sh
./install.sh
```
The script installs required packages and sets up configs.

Warning:
This will overwrite existing configs in your home directory.
Backup first if you care about your old setup.
